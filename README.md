# English

## Description
A dynamic image carousel built using HTML, CSS, and JavaScript. This project demonstrates JavaScript skills for looping through an array of objects and dynamically displaying content.

## Purpose
This project is a practice exercise for handling arrays and creating interactive carousels with JavaScript.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/IO-underscorre/js-array-objects-carousel.git
   ```
2. Open `index.html` in your browser.
